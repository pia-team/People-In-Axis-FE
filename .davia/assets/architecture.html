Architecture

Modern microservices-ready architecture with clear separation of concerns.

Frontend communicates with backend via REST API. Backend uses JPA for database access. Keycloak handles authentication.

System Layers:

  • Presentation: React frontend (Port 3000)
  • Application: Spring Boot backend (Port 8080)
  • Data: PostgreSQL database (Port 5432)
  • Security: Keycloak authentication (Port 8180)

Communication:

  • Frontend ↔ Backend: REST API (HTTPS)
  • Backend ↔ Database: JDBC (JPA/Hibernate)
  • Frontend ↔ Keycloak: OAuth2/OIDC
  • Backend ↔ Keycloak: JWT validation
