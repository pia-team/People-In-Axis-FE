Employee Module

Employee management system.

Features:

  • Employee CRUD operations
  • Profile management with photos
  • Department and company assignment
  • Manager-subordinate relationships
  • Employee code auto-generation
  • Excel import/export
  • Advanced search and filtering

Employee Lifecycle:

  • Create employee record
  • Assign to department and company
  • Set manager relationship
  • Update profile information
  • Track employment status
  • Archive when inactive

Access Control:

  • HUMAN_RESOURCES: Full CRUD access
  • ADMIN: Full access + system settings
  • Others: View own profile only

Data Fields:

  • Personal: Name, email, phone, address
  • Employment: Employee code, status, type
  • Organization: Company, department, manager
  • Metadata: Created date, updated date
