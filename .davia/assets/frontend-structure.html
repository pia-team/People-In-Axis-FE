Frontend Structure

Frontend code organization and file structure.

Directory Organization:

  • pages/: Route components (one per route)
  • components/: Reusable UI components
  • services/: API client functions
  • types/: TypeScript type definitions
  • hooks/: Custom React hooks
  • store/: Redux store and slices
  • layouts/: Page layout components

Component Patterns:

  • Functional components with hooks
  • Props interface definitions
  • Error boundaries for error handling
  • Loading states with React Query

Naming Conventions:

  • Components: PascalCase (PositionList.tsx)
  • Services: camelCase (positionService.ts)
  • Types: PascalCase interfaces (Position, Application)
  • Hooks: camelCase with "use" prefix (useKeycloak.ts)
