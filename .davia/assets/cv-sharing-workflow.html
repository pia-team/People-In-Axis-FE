CV Sharing Workflow

Role-based workflows for recruitment and talent management.

Key Features:

  • Position creation and publishing
  • Application submission with KVKK consent
  • CV pool management
  • AI-powered matching
  • Review and rating system

Access Control:

  • HUMAN_RESOURCES: Full access to all features
  • COMPANY_MANAGER: View-only, own CVs only
  • MANAGER: Position and application management
