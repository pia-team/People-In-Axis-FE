Project Module

Project management and tracking.

Features:

  • Project CRUD operations
  • Status tracking (Active, Completed, Cancelled)
  • Resource allocation
  • Project-employee relationships
  • Timesheet association

Project Workflow:

  • Create project with details
  • Assign to company
  • Set project status
  • Associate with timesheets
  • Track project progress

Access Control:

  • TEAM_MANAGER: Create and manage projects
  • HUMAN_RESOURCES: Full access
  • EMPLOYEE: View assigned projects

Integration:

  • Linked to timesheets for time tracking
  • Used in expense categorization
  • Shown in employee dashboards
