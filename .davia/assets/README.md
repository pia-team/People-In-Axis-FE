### Viewing Documentation

#### Option 1: Davia Local Server
```bash
davia open
```
Opens documentation in browser at `http://localhost:3000`

#### Option 2: GitHub
View on GitHub: `https://github.com/pia-team/People-In-Axis-FE/tree/unit-test/.davia/assets`

#### Option 3: Davia.ai Platform
Push to Davia.ai platform using:
```bash
davia push
```
