CV Sharing Module Details

Comprehensive CV sharing and application management.

Main Components:

  • Positions: Job postings with skills and requirements
  • Pool CVs: Talent pool with CV storage
  • Applications: Job applications with workflow
  • Matching: AI-powered CV-position matching

Position Management:

  • Create and publish positions
  • Define required skills and languages
  • Set status (Draft, Active, Passive, Closed, Archived)
  • View applications per position

CV Pool Features:

  • Store candidate profiles
  • Search by experience, skills, languages
  • Tag and categorize CVs
  • Match with positions

Application Processing:

  • 3-step application wizard
  • KVKK consent handling
  • File uploads (CVs, documents)
  • Status tracking
  • Rating and commenting system
  • Forwarding to reviewers
  • Meeting scheduling
