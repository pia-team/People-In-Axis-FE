Troubleshooting Guide

Common errors and solutions.

Debugging Steps:

  • Check browser console for frontend errors
  • Check backend logs: logs/application.log
  • Verify database connection
  • Verify Keycloak is running
  • Check network tab for API requests

Log Locations:

  • Backend: People-In-Axis-BE/logs/application.log
  • Frontend: Browser DevTools Console
  • Keycloak: Keycloak server logs

Useful Commands:

  • mvn clean install: Rebuild backend
  • npm run build: Check TypeScript errors
  • npm run dev: Start dev server with hot reload
  • davia open: View documentation

Health Checks:

  • Backend: http://localhost:8080/actuator/health
  • Frontend: http://localhost:3000
  • Keycloak: http://localhost:8180
  • Database: psql -U postgres -d people_in_axis
