Expense Module

Expense management and reimbursement workflows.

Features:

  • Expense recording with receipts
  • Category-based expenses
  • Approval workflow (Team Manager → Finance/HR)
  • Payment tracking
  • Excel import/export

Status Flow:

  • Draft → Submitted → Pending Manager
  • Approved by Manager → Pending Finance/HR
  • Paid/Approved → Completed

Access:

  • Employees: Create own expenses
  • Team Managers: Approve team expenses
  • Finance: Process payments
  • HR: Approve expenses
