Getting Started

Quick setup guide for developers.

Prerequisites

  • Java 21+
  • Node.js 18+
  • PostgreSQL 14+
  • Maven 3.8+
  • npm 9+ or yarn 1.22+
  • Keycloak 24.0+ (or access to instance)

Backend Setup

cd People-In-Axis-BE
mvn clean install
createdb people_in_axis
# Configure application.yml (database, keycloak)
mvn spring-boot:run

Backend runs on http://localhost:8080

Frontend Setup

cd People-In-Axis-FE
npm install
cp .env.example .env.local
# Configure .env.local (API URL, Keycloak URL)
npm run dev

Frontend runs on http://localhost:3000

Keycloak Configuration

  • Create realm: people-in-axis
  • Create client: people-in-axis-frontend
  • Set Valid Redirect URIs: http://localhost:3000/*
  • Define roles: ADMIN, HUMAN_RESOURCES, COMPANY_MANAGER, etc.

First Login

  • Open http://localhost:3000
  • Redirected to Keycloak login
  • Login with valid credentials
  • Access granted based on role
  • Dashboard displayed according to permissions
