Frontend State Management

State management strategy using Redux and React Query.

State Types:

  • Server State: React Query (caching, refetching)
  • Global State: Redux Store (auth, UI preferences)
  • Form State: React Hook Form (form inputs, validation)
  • Local State: useState (component-specific)

React Query Usage:

  • Automatic caching and refetching
  • Background updates
  • Optimistic updates
  • Error handling and retry logic

Redux Store Structure:

  • authSlice: Authentication state
  • uiSlice: UI preferences (theme, language)
  • queryClient: React Query cache
