Configuration Guide

Configuration files and environment setup.

Backend Configuration (application.yml):

  • spring.datasource: Database connection
  • keycloak: Authentication server settings
  • file.upload-dir: File storage location
  • logging: Log levels and patterns

Frontend Configuration (.env.local):

  • VITE_API_BASE_URL: Backend API URL
  • VITE_KEYCLOAK_URL: Keycloak server URL
  • VITE_KEYCLOAK_REALM: Keycloak realm name
  • VITE_KEYCLOAK_CLIENT_ID: Frontend client ID

Environment Profiles:

  • dev: Development settings (debug logging)
  • prod: Production settings (optimized)
  • cv-sharing: CV Sharing module config

Keycloak Configuration:

  • Realm: people-in-axis
  • Client: people-in-axis-frontend (public)
  • Client: people-in-axis-backend (confidential)
  • Valid Redirect URIs: http://localhost:3000/*
